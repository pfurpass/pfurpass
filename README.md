## 👻 A little about me...
[![GitHub WidgetBox](https://github-widgetbox.vercel.app/api/profile?username=pfurpass&data=followers,repositories,stars,commits&theme=dark)](https://github.com/Jurredr/github-widgetbox)

I am a **Full-Stack & Software Engineer** from **Austria** 🇦🇹 with a passion for building tools that live in the terminal, the browser, and everywhere in between. I love creating solutions that are both elegant and practical — from CLI utilities to full-blown web apps.

Currently I'm building open-source projects under the **Software-Hub** umbrella, focusing on developer tooling, remote desktop solutions, and database migration utilities.

At the moment I am diving deeper into systems programming and exploring how far I can push terminal-based UX — turning the command line into something fun and alive.
